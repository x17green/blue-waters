# Brand Logo Assets

## Required Files

Please save the following logo files to this directory:

1. **bayelsa-coat-of-arms.png** - Bayelsa State Government Coat of Arms
   - The colorful emblem with leopard, oil rig, palm tree, dolphin
   - Use: Homepage header (left position), official documents

2. **ministry-blue-economy-seal.png** - Ministry of Marine and Blue Economy Seal
   - The circular blue seal with boat and "BLUE ECONOMY, BAYELSA'S FUTURE" text
   - Use: Homepage header (right position), footer

## File Specifications

- **Format**: PNG with transparency preferred
- **Recommended sizes**:
  - Header logos: 120-150px height
  - Footer logos: 80-100px height
  - High DPI versions for retina displays

## Usage

These logos are referenced in the brand component system:
- `src/components/brand/logo-suite.tsx`
- Brand guidelines: `docs/branding.md`
